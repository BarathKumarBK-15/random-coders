Problem 1: Primo Round: Foreseen Betrayal

Problem Statement

Yes it's official! Kiruba is finally not leaving RANDOM CODERS. To celebrate this, Barath has arranged for a foreign trip. He was fascinated about the place called Sudan, so they planned to go there. Jayan Praveen who is also a part of the Designing Team was highly underrated compared to Rachit. So, he was very jealous of him and planned something for him. He arranged a survival game for Rachit in Sudan believing that Rachit won't survive it, so that he could become the only Designer for RANDOM CODERS. Rachit was pushed into primo round unwillingly. Now, the only way for him to survive is to complete game.

In the primo round, Rachit is given a stick of length L and is required to cut the stick into n equal parts, such that the length of each part is an integer greater than one. Print "YES", if Rachit could survive this game, else print "NO". Determine the fate of Rachit in primo round.

Note: The lengths of given stick is always non-prime.

Input Format

* The first line contains a single integer, t representing the no. of test cases.
* The first line of each test case contains a single integer, l representing the length of the stick given.

Constraints

* 1 <= t <= 100
* 1 <= l <= 10 ^ 16
* l is always a non-prime number.

Output Format

* For each test case, print "YES" if the stick can be divided into n equal parts with length of each part is greater than one, else print "NO" (without quotes).

Sample Input 0

2
9
12

Sample Output 0

YES
YES

Explanation 0

* Case 1: The stick of length 9 can be broken into 3 qual parts each of length 3.Thus print 'YES'

* Case 2: The stick of length 12 can be broken in multiple ways such as 2 parts each of length 6 units, 3 parts each of lenght 4 units and so on. Thus, print "YES"

Problem 2: Primo Round : Rachit's Secondo Escape

Problem Statement

Rachit, by luck figured it out in the primo round. But Jayan Praveen was more smarter than him. So, if he could break the stick of length l into n equal parts such that the length of each part is an integer greater than one, he ordered Rachit to find the length at which the value of n is maximised.

Hint: For n to be maximised, the length of each broken part of the stick must be as minimum as possible.

Note: The lengths of given stick is always non-prime.

Input Format

* The first line contains a single integer, t representing the no. of test cases.
* The first and the only line of each test case contains a single integer, l representing the length of the given stick (which is always non-prime).

Constraints

* 1 <= t <= 100
* 1 <= l <= 10 ^ 8
* l is always a non-prime number.

Output Format

* For each test case print a single integer greater than one which represents the length of broken part at which the value of n is maximised.

Sample Input 0

2
25
12

Sample Output 0

5
2

Explanation

* Case 1: The stick of length 25 can only be cut equally as 5 parts each of length 5 units. thus print '5' * Case 2: The stick of length 12 can be cut in various ways as, 2 parts of length 6 each, 3 parts of length 4 each, 4 parts of length 3 each and 6 parts of lenght 2 each. Thus the maximum number of pieces is obtained when length of each part is 2. Thus print '2'.

Problem 3: Terzo Round : Rachit's Narcos Addiction

Problem Statement

The earlier rounds were piece of cake for Rachit since it just invloved logical thinking. So, this time Jayan Praveen wanted to test his physical endurance. There are N hurdles of heigths variying from 1 to N present in an array with 1-based indexing. Jayan Praveen chooses a set of hurdles from the above starting from index a to b (a <= b, both inclusive). Rachit knows that he would not be able to clear the task and thereby decides to consume drugs. The quantity of drugs consumed must be equal to the total height that Rachit must jump to clear this challenge. Consuming drugs more than the required quantity may end up in severe issues, on the other hand consuming less than the required will be a barrier for Rachit to clear this challenge. Help him find the exact quantity of drugs that he must consume.

Note 1: To clear a hurdle, Rachit must jump at the height exactly one greater than the height of the hurdle.

Note 2: The value of drugs consumed may be very high, therefore compute it modulo 10^9+7.

Input Format

* The first line contains a single integer, t representing the no. of test cases.
* The first line of each test case contains a single integer, N representing the no. of hurdles.
* The second line of each test case contains two space seperated integers, a and b representing the starting and ending index of the set of hurdles chosen by Jayan Praveen.

Constraints

* 1 <= t <= 100
* 1 <= N <= 10 ^ 9
* 1 <= a <= b <= N
* Sub Task 1( 50 points ): 1 <= N <= 10^6
* Sub Task 2( 100 points ): Original Constraints

Output Format

* For each test case, print a single integer representing the amount of drugs modulo 10^9+7 to be consumed by Rachit to clear this challenge safely.

Sample Input 0

1
5
1 4

Sample Output 0

14

Explanation 0

* The choosen hurdles are of length 1, 2, 3 and 4. therfore Rachit has to jump heights 2, 3, 4 and 5. Therfore total height to be jumped by Rachit is 14. Thus quantity of drug to be consumed is 14

Problem 5: Quinto Round : Smoke Shisha Play Fifa

Problem Statement

It's time for the Final Showdown. Now Jayan Praveen, wanted Rachit to survive a game based on mere luck. The final challenge is a football game between Offenders and Defenders. There are always four defeners, each with some specific energia. For example

40 50 50 40
D1 D2 D3 D4

The formation for the Offenders consists of three players, with their own energia.For example

60 45 55
F1 F2 F3

In any situation, for any Offender to surpass the defense, his energia must be strictly greater than that of the two nearest defenders.

In the above situation, F1 can surpass the defence as his energia is strictly greater than his nearest defenders ( D1 and D2 ). Similarly, F3 will be able to surpass the defense.

However, F2 wont be able to surpass the defense as the energia of D2 and D3 are greater than his.

Rachit, has N Offenders with their own Energia, among which he can choose any three players to surpass the defense, randomly.
Rachit, wants to survive the final showdown and thereby asks Barath and Kiruba to find the probablity that he will win the game. If the probablity of him winning is P/Q.

Find P and Q such that P/Q is in the most possible simplified ratio.

Note: Rachit wins the game if all three player would be able to surpass their corresponding two defenders

Input Format

* The first line contains a single integer T, representing the number of test cases
* The first line of each test case contains 4 space seperated integers, representing the energia of the defenders.
* The second line of each test case contains a single integer N, representing the number of Offenders available
* The third and the last line of each test case contains N space seperated integers representing the Energia of corresponding Offender.

Constraints

* 1 <= T <= 100
* 1 <= D[i] <= 100
* 3 <= N <= 1000
* 1 <= F[i] <= 100

Output Format

* For each test case, print 2 space seperated integers, representing the values of P and Q, such that P/Q is the most simplified ratio

Sample Input 0

1
30 40 60 50
4
65 65 45 35

Sample Output 0

1 12

Explanation 0

* 3 Players out of the given four players can be chosen randomly in 24 different ways.
* If the given four players are considered P1, P2, P3 and P4 respectively. Out of all the formations only 2 formations, P3 P1 P2 and P3 P2 P1 will be able to surpass the defense. Thus the probability of winning is 2/24, whose simplified form is 1/12.
* Thus print, "1 12"
