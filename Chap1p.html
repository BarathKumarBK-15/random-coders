Problem 1: Munchy and Odd Parity

Problem Statement

Munchy recently got an array of N elements. Since he is happy about Dazzler's breakup with his girlfriend , he decided to do something "Odd" with the array. Help him pick any two numbers A and B from the array such that the sum of A and B is odd. If such a possible situation is there print YES. If not print NO.

Input Format

* The first line is T consisting number of testcases.
* The next line contains N denoting the number of elements of the array.
* Then next line consists of N space-separated integers of the array.

Constraints

* 1<=T<=10
* 2<=N<=100000 0

Output Format

* For each test case, print either YES or NO.

Sample Input 0

2
5
4 8 100 3 1
2
7 101

Sample Output 0

YES
NO

Explanation 0

* 1.In the first case he can pick 3 and 100 and the sum 103 is odd.
* 2.In the second case 7 + 101 gives 108 which is even.

Approach

* Addition of any even number with an odd number will always give an odd number. Count the number of even and odd numbers in the array.
* If both the counts are greater than 0 , then it is possible else not.

Code in Python

from math import *
from collections import *
from random import *
from bisect import *
import sys
input=sys.stdin.readline
t=int(input())
while(t):
    t-=1
    n=int(input())
    r=list(map(int,input().split()))
    s='NO'
    e,o=0,0
    for i in r:
        if(i%2):
            e=1
        else:
            o=1
    if(o&e):
        s='YES'
    print(s)

Code in C++

    }

}

Problem 2: Dazzler and his Ego

Problem Statement

As Munchy insulted Dazzler in his above question, Dazzler divided to take revenge on him. He gave Munchy this problem and asked him to solve. Help Munchy in solving this problem

You are given two strings A and B of equal length N. You can do the following operation any number of times.You can select any character from A and swap it with any character in B. Formally, Select any i(1<=i<=N) and j(1<=j<=N), You can swap Ai and Bj. Your task is to make both A and B equal, ie.. for any i(1<=i<=N) Ai=Bi. Check whether it is possible or not. If possible print "YES"(without quotes), else "NO"(without quotes)

Input Format

* First line will contain T, number of testcases.
* Then the testcases follow. The first line of each test case contains N.
* The second line of each test case contains A.
* The third line of each test case contains B.

Constraints

* 1<=T<=100
* 1<=N<=100000

Output Format

* print a single line for each test case, "YES"(without quotes) if it is possible or "NO"(without quotes) if it is not possible

Sample Input 0

2
5
ABCDE
ABCDE
5
ABCDE
FGHIJ

Sample Output 0

YES
NO

Approach

* We can make it possible if the count of each alphabet in both the strings is even. For Example, if A="ABCDE" and B="ABCDE", each characters occur twice, hence it is possible.

Code in Python

t=int(input())
for _ in range(t):
    n=int(input())
    a=input()
    b=input()
    #count array keeps the count of each alphabet in both a and b
    count=[0 for i in range(26)]
    for i in a:
        count[ord(i)-65]+=1
    for i in b:
        count[ord(i)-65]+=1
    flag=1
    #next iteration loop checks for any odd occurence of a character
    for i in count:
        if(i%2!=0):
            flag=0
            break
    if(flag==0):
        print("NO")
    else:
        print("YES")

Problem 3: Munchy vs Dazzler

Problem Statement

This time the fight between Munchy and Dazzler become more ferocious. So they decided to play this game. This time their friends Kiruba and BK15 joined them. Kiruba and Dazzler team up, Munchy and BK15 team up.

Initially, there will be an array A of N elements. N is always even. A contains only non-negative integers. Each player gets an alternate chance. The game starts with Munchy's Team chance, then Dazzler's Team and it continues. The game ends with Dazzler's play. The integer M and D denotes the scores of Munchy and Dazzler respectively. Initiall M and D will be zero. Each time Munchy picks an element, he will do any one of the following according to the condition:

1.if A[i] is even make M = (A[i]*A[i])-M.

2.if A[i] is odd then make M = floor((M+A[i])/2).

Each time Dazzler picks an element, he will do any one of the following according to the condition:

1.if A[i] is even make D = (D+(3*A[i])).

2.if A[i] is odd then make D = floor((D*A[i])/(D+A[i]+1)).

After picking all the elements, the game ends.
If D>M, print "DAZZLER and KIRUBA WILL WIN" (without quotes)
else if M>D, print "MUNCHY and BK15 WILL WIN" (without quotes),
else print "TIE" (without quotes).
Also print their scores

Input Format

* First line will contain T, number of testcases.Then the testcases follow.
* The first line of each test case contains N.
* The second line of each test case contains the array A.

Constraints

* 1<=T<=100000
* 2<=N<=100000
* 0<=A[i]<=100000
* Sum of N over all test cases does not exceed 100000

Output Format

* For each testcase print two lines.
* In the first line, print "DAZZLER and KIRUBA WILL WIN" (without quotes)if D>M, else print "MUNCHY and BK15 WILL WIN"(without quotes), else print "TIE" (without quotes)
* In the second line, print the score of Munchy's team and Dazzler's team respectively

Sample Input 0

3
4
1 2 3 4
4
11 0 11 0
6
0 0 0 0 0 0

Sample Output 0

DAZZLER and KIRUBA WILL WIN
1 18
MUNCHY and BK15 WILL WIN
8 0
TIE
0 0

Explanation 0

* Initially M and D will be Zero. The game starts with Munchy's team. They select the first element ie.. 1
* 1 is an odd number so do the second operation M=floor((M+A[i])/2), so after first move it remains the same as floor(1/2)=0
* Next, Dazzler's team pick the second element ie.. 2
* 2 is an even number so do the first operation D=D+(3*A[i]), so after the second move D will be equal to 6
* The game continues.. at the end M and D will be equal to 1 and 18 respectively. So Dazzler's team wins the game

Approach

- Follow the operations given in the question and calculate M and D.
- If M == D, then print TIE.
- If M > D, then print MUNCHY and BK15 WILL WIN.
- If M < D, then print DAZZLER and KIRUBA WILL WIN.
- In the next line print M and D respectively.

Code in Python

import math

t=int(input())
for k in range(t):
    n=int(input())
    arr=list(map(int,input().split()))
    d,m=0,0
    for i in range(n):
        if i%2:
            if arr[i]%2:
                d=math.floor((d*arr[i])/(d+arr[i]+1))
            else:
                d+=3*arr[i]
        else:
            if arr[i]%2:
                m=math.floor((arr[i]+m)/2)
            else:
                m=arr[i]*arr[i]-m
    if d>m:
        print("DAZZLER and KIRUBA WILL WIN")
    elif m>d:
        print("MUNCHY and BK15 WILL WIN")
    else:
        print("TIE")
    print(m,d)

Code in C++

    int t;
    cin >> t;
    while(t--) {
        int n;

        long long M = 0;
        long long D = 0;
        for(int i = 1; i <= n; i++) {
            int element;
            cin >> element;
            if(i % 2 == 0) {
                if(element % 2 == 0) {
                    D = 3 * element + D;
                }
                else {
                    D = (D * element) / (D + element + 1);
                }
            }
            else {
                if(element % 2 == 0) {
                    M = element * element - M;
                }
                else {
                    M = (M + element) / 2;
                }
            }
        }
        if(D == M) {
            cout << "TIE" << endl;
            cout << M << " " << D << endl;
        }
        else if(M > D) {
            cout << "MUNCHY and BK15 WILL WIN" << endl;
            cout << M << " " << D << endl;
        }
        else {
            cout << "DAZZLER and KIRUBA WILL WIN" << endl;
            cout << M << " " << D << endl;
        }
    }
    return 0;
}

Problem 4: Munchy-Dazzler-Assemble

Problem Statement

Munchy and Dazzler,after a fight finally decided to become good friends since they need to prepare for ICPC.

But to make that possible they have to build one palindrome each consisting of exactly x number of a's and y number of b's.

Each one has to build a palindrome different from the other one satisfying the above conditions.

If they cannot build two distinct palindromes then print -1.

If there are multiple solutions possible print the two strings in any order.

Our checker solution will validate it(no harm huh?).

Input Format

* First line consists of x - the number of a's
* Second line consists of y - the number of b's

Constraints

* 1<=x<=10000000
* 1<=y<=10000000

Output Format

* Print the two distinct palindromes in any order or -1 if its not possible.

Sample Input 0

2
2

Sample Output 0

abba
baab

Approach

There are many approaches to solve this problem. Let me explain my following approach.
First of all a palindrome must contain all letters an even number of times, except the middle letter(if length of palindrome is odd, this is ignored in even length).

We deal with only two characters here **a** and **b**.Let **x** and **y** be number of **a's** and **b's** respectively. If x and y are odd then its impossible to do so, since both letters are occuring odd number of times and the total length of the palindrome will be even.

If either x or y is 1,then the position of one character is fixed, so we can't obtain two distinct palindromes.

Else it's always possible to obtain a valid solution.

If x and y both are even, then we can simply split into three components,
For example if x is 4 and y is 6
We can split it into 'aa' and 'bbbbbb' and 'aa'.
Similarly we can split into 'bbb' and 'aaaa' and 'bbb'.
The proof is quite simple and is left upto you.

If x is even and y is odd then we can split into five components,
For example if x is 4 and y is 5
We can split into 'aa' and 'bbbbb' and 'aa'.
Then the next split can be 'bb' and 'aa' and 'b' and 'aa' and 'bb'.
Notice 'b' takes the middle character since y is odd.
Similarly you can do for the other case too.

Code in Python

from math import *    
from collections import *     
from random import *     
from bisect import *     
import sys

t=1

    t-=1

    y=int(input())
    if((x%2 and y%2) or (x==1 or y==1)):
        print(-1)
        exit(0)
    elif(x%2==0 and y%2==0):
        r1=('a'*(x//2))+('b'*y)+('a'*(x//2))
        r2=('b'*(y//2))+('a'*x)+('b'*(y//2))
    elif(x%2):
        r1=('a'*(x//2))+('b'*(y//2))+('a')+('b'*(y//2))+('a'*(x//2))
        r2=('b'*(y//2))+('a'*x)+('b'*(y//2))
    else:
        r1=('b'*(y//2))+('a'*(x//2))+('b')+('a'*(x//2))+('b'*(y//2))
        r2=('a'*(x//2))+('b'*y)+('a'*(x//2))
    print(r1)
    print(r2)
