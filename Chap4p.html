Problem 1: Munchy's Sizzling Cakewalk!

Problem Statement

Before the storyline continues,Munchy as usual left a strong-stronger-strongest question for the people.
Munchy has N candies and he needs to divide it among M girls.Of course the distribution must satisfy the following conditions :
1.Each girl must receive an even number of candies.
2.Each girl must receive at least one candy.
Find whether it's possible or not.
NOTE THAT YOU DON'T HAVE TO NECESSARILY GIVE ALL CANDIES, YOU CAN KEEP SOME OF THEM.

Input Format

* The first line consists of T denoting the number of testcases.
* Each line conatains 2 space-separated integers N and M.

Constraints

* 1<=T<=100000
* 1<=N<=10^18
* 1<=M<=10^18

Output Format

* Print YES ,if it's possible or else NO.

Sample Input 0

1
6 2

Sample Output 0

YES

Explanation 0

* 1st girl can receive 4 candies and 2nd girl can receive 2 candies.

Approach

- It is sufficient to give 2 candies to each girl.
- So check if 2*M is within N or not.

Code in Python

t=int(input())
while(t):
    t-=1
    n,m=map(int,input().split())
    s='NO'
    if(2*m<=n):
        s='YES'
    print(s)

Problem 2: Munchy's Perfect Palindromes

Problem Statement

Munchy was expecting Dazzler to survive his traps. He didn't want anymore fight with him and decided to seek peace and become good friends.So he framed a perfect friendly problem for Dazzler.Help him solve it.

You are given a binary string S.You can do the following operation at most K (possibly zero)times:

1.If the bit is '0', change it to '1'.
2.Else if the bit is '1', change it to '0'.

Find out whether it's possible to make a palindrome by performing at most K operations.

Input Format

* The first line consists of T denoting the number of testcases.
* Second line consists of 2 space-separated integers**N** and K denoting the length of the binary string and the limit of the operations,respectively.
* Third line consists of S, the binary string.

Constraints

* 1<=T<=100
* 1<=|S|<=100000
* 1<=K<=|S|

Output Format

* Print a single line for each testcase, YES, if it's possible or else NO.

Sample Input 0

2
3 1
001
2 0
01

Sample Output 0

YES
NO

Explanation

1.We can flip the 1st bit to '1' using one operation and the string becomes 101,which is a palindrome.
2.In this case it's impossible to make it a palindrome using 0 operations.

Approach

Divide the string into equal halves , ignore the middle element if it's of odd length.
Let i represent the first ith element in the first half.
Let j represent the first jth element from the last in the second half.
Now for a palindrome ith and jth character must be same.
If not just flip any one bit that's enough.
Continue for the remaining N//2 characters.
If the total number of flips required is more than K,then its not possible else its possible.

Code in Python

t=int(input())
while(t):
    t-=1
    n,k=map(int,input().split())
    s=input().rstrip('\n')
    co=0
    r='YES'
    for i in range(n//2):
        if(s[i]!=s[n-i-1]):
            co+=1
    if(co>k):
        r='NO'
    print(r)

Problem 3: Dazzler and his Even Pairs for Munchy

Problem Statement

As last time it was easy to find the odd pairs.. Dazzler and Munchy solved the problem easily. But this time it is'nt the case. They have to find the number of even pairs. Help them to solve the problem.

Read the problem carefully and also the note.
You are given with an array A of length N consisting of non negative integers. You have to count the number of pairs such that a[i]+a[j] is an even integer, where 1<=i<=N, 1<=j<=N.

Note : i should not be equal to j. If you are a C++ user , use long long and not int.

Input Format

* The first line contains T which indicates the number of testcases.
* The first line of each test case contains N.
* The second line of each test case contains the array of length N

Constraints

* 1<=T<=1000
* 2<=N<=1000000
* 0<=A [ i ]<=10000000
* The sum of N over all cases does not exceed 1000000

Output Format

* For each test case print a single line indicating the number of pairs

Sample Input 0

2
2
1 0
2
3 3

Sample Output 0

0
2

Explanation 0

* In the first testcase its not possible to form an even pair.
* In the second testcase a[ 0 ]+a[ 1 ] = 6 and a [ 1 ]+a [ 0 ] = 6, so its 2.

Approach

Number of even pairs will be odd*(odd-1) + even*(even-1), where odd and even are the count of odd and even integers respectively

Code in Python

t=int(input())
for _ in range(t):
    n=int(input())
    a=list(map(int,input().split()))
    even,odd=0,0
    for i in a:
        if(i%2==0):
            even+=1
        else:
            odd+=1
    print(even*(even-1)+odd*(odd-1))

Problem 4: Dazzler's Munchful Array

Problem Statement

Dazzler wants to gift an array to Munchy. Before that he asks Munchy what type of array you need. Munchy asked Dazzler to make his array Munchful

An array is called Munchful if all its elements are divisble by k.

You are given an array A of N elements. You can make any one of the following operation any number of times

* Make a[ i ] as a[ i ] + 1
* Make a[ i ] as a[ i ] - 1

Each first operation costs F dollars and each second operation costs S dollars.

You have to make all the elements of the array divisble by k. If B is the array after required operations are done, then b [ i ] % k should be zero 1<=i<=N.

Find minimum how many dollars are required to make the array munchful.

Since the answer may be huge, compute it modulo 10^9+7

Input Format

* The First line contains T, number of Testcases.
* The First line of each Test case contains four integers indicating N,K,F,S respectively.
* The second line of each Test case containing N spaced integers containing the array A.

Constraints

* 1<=T<=100
* 1<=N<=100000
* 0<=A [ i ]<=100000
* 1<=k<=100000
* 0<=F<=1000
* 0<=S<=1000
* Sum of N over all testcases does not exceed 5.10^5

Output Format

* For each test case, print a single line that contains minimum dollars required to make the array munchful modulo 10^9+7

Sample Input 0

1
3 2 1 1
1 1 1

Sample Output 0

3

Approach

- for each element to make it divisble by k the minimum cost must be min((a[i]%k)*s,(k-a[i]%k)*f) ,or each 1<=i<=N

Code in Python

t=int(input())
for _ in range(t):
    n,k,f,s=map(int,input().split())
    a=list(map(int,input().split()))
    mod=[0 for i in range(n)]
    for i in range(n):
        mod[i]=a[i]%k
    ans=0
    for i in mod:
        gpay=k-i
        ans=(ans+min(gpay*f,i*s))%(10**9+7)
    print(ans)

Code in C++

#include<iostream>
#define mod 1000000007
using namespace std;
int main()
{
    int t;
    cin>>t;
    while(t--)
    {
        int n,k,f,s,sum=0;
        cin>>n>>k>>f>>s;
        while(n--)
        {
            int num;
            cin>>num;
            if(num%k!=0)
            {
                int x=num-(num/k)*k;
                int y=(num/k+1)*k-num;
                sum=(sum+min(y*f,x*s))%mod;
            }
        }
        cout<<sum<<endl;
    }
    return 0;
}

Problem 5: Munchy's Dazzling Bits!

Problem Statement

Munchy is bored of the mechanical life and decided to do something interesting rather than attending classes.He and Dazzler decided to improve their programming skills.

Then,one fine day there came a secret mail to them.When Munchy opened it , it wasn't surprising for him since it was a RANDOM challenge sent to him by someone.

He was curious and wanted to solve the challenge,but couldn't.And he saw the mail was sent by someone named Lukaku.Who is Lukaku?
What happened in Munchy's and Dazzler's past life? Before they can answer these..Help them solve this!

You are given an array of N elements. Let's define a function F over two numbers A and B as follows:

1.Do OR operation between A and B and let the resultant be R(in decimal representation).

Now find the sum of F(A,B) where (A,B) are the possible unordered pairs from the given array elements modulo 10^9+7.

Input Format

* The first line consists of T denoting the number of testcases.
* The second line contains N denoting the number of elements in the array.
* The third line contains N space-separated integers of the array.

Constraints

* 1<=T<=10
* 1<=N<=100000
* 0<=Ai<=10^9

Output Format

* Print the required sum modulo 10^9+7.

Sample Input 0

1
3
1 2 3

Sample Output 0

9

Explanation 0

* F(1,2)=3 F(1,3)=3 F(2,3)=3 Hence total sum would be 3+3+3=9

Approach

- There will be at maximum 30 bits since maximum value can be 10^9.So maintain a bit array of size 30.
- 0th position means 2^29 is included in binay representation
- 29th position means 2^0 is included in binary representation.
- For each element if ith bit is set then increment the ith position of bit array.
- Finally let's say we for ith position we have **y** number of ones .We can choose
- n*(n-1)/2 pairs for ith bit. Out of them we only need one bit at least to be set, so lets find out how many pairs can be taken such that (0,0) bit is satisfied.
- There will be N-y number of zeroes and let it be 'x'. And x*(x-1)/2 pairs can be taken where (0,0) pairs are satisfied.
- Hence subtract it from total pairs and multiply it with its 2 power.
- Do this for each bit position.
- Dont forget to take modulo!!!

Code in Python

t=int(input())
while(t):
    t-=1
    p=1000000007
    n=int(input())
    a=list(map(int,input().split()))
    b=[0]*30
    for i in a:
        z=bin(i)[2:]
        z=z.zfill(30)
        for j in range(30):
            if(z[j]=='1'):
                b[j]+=1
        r=0
    for i in range(30):
        y=n-b[i]
        re=((n*(n-1))//2)-((y*(y-1))//2)
        r+=re*(1<<(30-i-1))
        r=r%p
    print(r)
